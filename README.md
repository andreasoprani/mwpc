# Milky Way Pool Club

This is a small game that I developed as a follow up to [Pikuma's course on 2D Game Physics Programming](https://pikuma.com/courses/game-physics-engine-programming), so the physics system is heavily derived from there.

It's basically a pretty basic pool game with gravity as a twist, nothing special.

The game is written in C using [raylib](https://www.raylib.com/).

The game assets are from [LocalThunk](https://localthunk.com/), taken from the planet cards in Balatro.
I can remove or replace them if this is a problem, I just liked the sprites a lot.

You can play it at [sprn.it/mwpc](https://sprn.it/mwpc).
